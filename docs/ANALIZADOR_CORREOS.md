# Analizador de correos de FARO v5

FARO v5 agrega una revisión simple de correos:

- remitente;
- dominio del remitente;
- entidad que dice ser;
- asunto;
- cuerpo del correo;
- pedidos peligrosos;
- escenarios típicos de phishing.

La lista interna incluye referencias conservadoras para Argentina y servicios muy usados:

- ARCA / ex AFIP;
- ANSES;
- Banco Nación;
- Santander;
- Galicia;
- Macro;
- Mercado Pago;
- Correo Argentino;
- Personal / Flow / Telecom;
- Microsoft / Outlook / Hotmail;
- Google / Gmail;
- Netflix.

Regla principal: que un dominio parezca oficial no significa que el correo sea seguro. Si pide claves, códigos, pagos, tarjeta, instalar programas o actuar urgente, FARO debe alertar.
