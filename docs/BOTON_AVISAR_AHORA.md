# Botón AVISAR AHORA

FARO v5.2 agrega un botón grande visible cuando la llamada llega a riesgo crítico.

El botón:

1. guarda la llamada;
2. registra el análisis;
3. abre WhatsApp Web con mensajes preparados;
4. usa contactos marcados como `extreme`.

FARO no presiona enviar de forma oculta. La persona revisa y confirma.
