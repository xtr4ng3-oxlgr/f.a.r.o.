# Seguridad

FARO es una herramienta defensiva de prevención y orientación.

## Reportar problemas

Para reportar errores o problemas de seguridad, contactar públicamente por:

```text
Facebook: https://www.facebook.com/Str0ma3/
```

## Principios de seguridad

FARO no debe:

- enviar mensajes ocultos;
- robar datos;
- espiar conversaciones;
- abrir links sospechosos automáticamente;
- capturar contraseñas;
- automatizar acciones invisibles;
- funcionar como malware o spyware.

## Privacidad

FARO trabaja localmente y no vende datos.

Si compartís reportes o capturas para ayudar a mejorar el proyecto, borrá antes:

- números reales;
- nombres completos;
- direcciones;
- documentos;
- correos personales;
- datos bancarios.
